## 📋 Описание

TelegramPC Guardian - это инструмент удаленного управления компьютером через Telegram бота. Позволяет контролировать ПК, получать информацию о системе, делать скриншоты, записывать аудио и многое другое.

## ✨ Возможности

- 📊 Получение информации о системе (имя ПК, IP-адреса, ОС)
- 📸 Создание скриншотов экрана
- 📷 Фото с веб-камеры
- 🎤 Запись аудио с микрофона
- ⌨️ Кейлоггер (логирование нажатий клавиш)
- 📋 Получение содержимого буфера обмена
- 🎵 Воспроизведение аудио файлов
- 🖼️ Показ изображений на весь экран
- 🎬 Воспроизведение видео
- 🌐 Мониторинг открытия определенных сайтов
- 🚀 Автозапуск при старте системы

## 🛠️ Установка

### Требования

- Python 3.8+
- Windows OS
- Telegram бот токен

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/DarkFimoz/TelegramPC-Guardian.git
cd TelegramPC-Guardian
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте бота:
   - Откройте `remote_control.py`
   - Замените `BOT_TOKEN` на токен вашего бота (получите у [@BotFather](https://t.me/BotFather))
   - Замените `CHAT_ID` на ваш Telegram ID (узнайте у [@userinfobot](https://t.me/userinfobot))

4. Запустите программу:
```bash
python remote_control.py
```

## 📱 Команды бота

- `/start` - Список всех команд
- `/info` - Информация о системе
- `/screenshot` - Сделать скриншот
- `/photo` - Сделать фото с камеры
- `/record <секунды>` - Записать аудио (по умолчанию 10 сек)
- `/keylog_start` - Запустить кейлоггер
- `/keylog_stop` - Остановить кейлоггер и получить лог
- `/keylog_get` - Получить текущий лог без остановки
- `/get_clipboard` - Получить содержимое буфера обмена
- `/stop_music` - Остановить воспроизведение музыки

### Работа с медиа

- Отправьте голосовое сообщение - автоматически воспроизведется
- Отправьте аудио файл с подписью `/start_music` - воспроизведет музыку
- Отправьте фото с подписью `/start_photo <секунды>` - покажет на весь экран
- Отправьте видео с подписью `/start_video` - воспроизведет видео

## 🔨 Сборка в EXE

Для создания исполняемого файла используйте `build.py`:

```bash
python build.py
```

Скомпилированный файл появится в папке `dist/`.
