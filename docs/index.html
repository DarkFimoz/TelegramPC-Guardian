🛠️ Установка

1

Клонируйте репозиторий

git clone https://github.com/DarkFimoz/TelegramPC-Guardian.git
2

Установите зависимости

pip install -r requirements.txt
3

Настройте бота

Замените BOT_TOKEN и CHAT_ID в remote_control.py

4

Запустите программу

python remote_control.py

📱 Команды бота

/start

Список всех команд

/info

Информация о системе

/screenshot

Сделать скриншот

/photo

Фото с камеры

/record

Записать аудио

/keylog_start

Запустить кейлоггер

/keylog_stop

Остановить кейлоггер

/get_clipboard

Буфер обмена

/stop_music

Остановить музыку
